AL TUO FIANCO - PROTOTIPO APP

Come provarla subito:
1) Estrai lo ZIP.
2) Apri index.html con Chrome, Edge o Safari.
3) Compila una richiesta: verrà salvata nel browser in locale.

Cosa contiene:
- Home servizi
- Prenotazione accompagnamento / pratica
- Upload documento demo
- Pannello richieste ricevute demo
- Collegamento WhatsApp da personalizzare
- Manifest PWA per futura installazione come app

Da personalizzare:
- Numero WhatsApp nel file index.html: sostituire 393000000000 con il tuo numero.
- Nome, colori, comuni serviti, prezzi.

Per una versione online reale serviranno:
- Database
- Login operatore/familiari
- Upload documenti sicuro
- Email/WhatsApp automatiche
- Pagamenti online
